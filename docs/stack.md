# 기술 스택

### 백엔드
- **프레임워크**: NestJS (TypeScript)
- **ORM**: TypeORM
- **데이터베이스**: PostgreSQL 16
- **캐시/세션**: Redis 7
- **인증**: JWT + Passport
- **API 문서**: Swagger
- **테스트**: Jest, Playwright

### 프론트엔드
- **프레임워크**: React 18 + TypeScript
- **빌드 도구**: Vite
- **라우팅**: React Router
- **상태관리**: Zustand
- **HTTP 클라이언트**: Axios
- **스타일링**: Tailwind CSS
- **테스트**: Jest, React Testing Library

### 인프라
- **컨테이너**: Docker + Docker Compose
- **웹서버**: Nginx
- **CI/CD**: GitHub Actions
